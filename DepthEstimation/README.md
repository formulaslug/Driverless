# Lite Mono archive
The model was deemed 'not good enough' to continue use. It is too slow, and just not good enough at large depths
Clone 
```
git@github.com:noahzn/Lite-Mono.git
```
into this folder and run 
```
python main.py
```
in order to get depth estimation
